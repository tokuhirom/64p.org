*PSGI
----
What's PSGI?
----
Perl Webserver Gateway *Protocol*
----
>||
lighty/apache/H::S::S/etc.
       ↓    ↑
  [ PSGI Protocol ]
       ↓    ↑
Catalyst/CGI app/etc.
||<
----
Protocol is very simple
- pass the primitive.
-- arrayref/hashref/string/integer/etc.
----
Implementer pass the $env(hashref)
----
$env contains CGI vars
- SERVER_NAME
- PATH_INFO
- QUERY_STRING
- etc.
----
